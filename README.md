# Actions-OpenWrt-AX6S 自用编译版

区分OpenWrt和lede版本，lede版本有MU-MIMO，目前以lede版本为主

默认带外部源openclash和adguardhome

建议关闭adguardhome统计功能，会不断占用空间

设置adguardhome作为dnsmasq的上游服务器或者 (重定向53)

上游DNS设置为 >> 127.0.0.1:7874（clash的dns端口）

main为保守稳定版，Plus为测试版
